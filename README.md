# frontend

靴磨きの店舗を探すアプリのfrontendのプロジェクトです

## ディレクトリ構成
```

src/src
|-app         store,rootReducer,全体で使用する定数など
|-assets      スタイルシート
|-components  UI
|-containers  UIで使用する stateとactionを作成しprops化する
|-images      ロゴなど
|-modules     Reduxソース
|-types       共通使用する型
|-utils       共通使用する関数
|-index.tsx   Reactエントリポイント
```
